【題名】
　打ち上げ花火シミュレーター

【操作方法】
　●キーボード操作
　　fキー：花火を1発打ち上げる（同時に花火30発まで描画可能）
　　sキー：花火の爆発モード切替（ランダム爆発/球座標爆発）
　●マウス操作
　　ドラッグすることで視点座標を移動させることができます。

【説明文】
　打ち上げ花火が持つ性質のうち、「爆発するまでどんな花火かわからない」という隠蔽性を作品に取り入れました。任意の数だけ発射された花火は、それぞれ爆発するまでどんな色・形をしているのか誰にも分かりません。

【解説】
　夏休みの一大イベントといえば「花火大会」です。この作品は、花火大会の主役である「打ち上げ花火」を再現したものになります。

　花火大会はなぜ観ていて飽きないのでしょうか？その答えは「次の花火はどんな色・形だろうか？」という疑問と期待を私たちに抱かせるためだと考えられます。つまり、打ち上げ花火には「爆発するまでどんな花火か分からない」という隠蔽性が存在することになります。
　この作品は、前述の花火が持つ性質に着目して制作されています。まるで、本物の花火大会を鑑賞しているかのような「ワクワク感」を楽しむことができます。
